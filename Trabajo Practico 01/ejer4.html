INTRODUCCION HTML

Este es un curso que pretende introducir al lector en el mundo del
HTML. No se requiere conocimientos previos, se empezara desde cero.
No hay que saber nada de programacion para utilizar HTML, basta
hasta aprender un par de cosas sensillas para empezar a crear paginas.

Es MUY FACIL aprender HTML. Tan solo necesitas un editor de texto, un
curso de html y navegador web (como netscape o internet explorer),
ni siquiera necesitamos estar conectado a internet.
